# 深度學習的技術 — 每日學習提示（每日摘要專用）

每日從楊大輝《深度學習的技術》五層學習框架中選取一則實用技巧。

## 執行方式

不需呼叫外部 API。根據今天的日期（星期幾）輪替選取，與原子習慣提示錯開主題。

## 提示庫

### 星期一：提取練習
主題：主動回憶比被動複習更有效
提示：學完新東西後，蓋上筆記嘗試回憶。問自己「我能想起什麼？」而非「讓我再看一次」。每次主動回想都在強化記憶迴路。
層級：記憶層（Level 1）

### 星期二：費曼技巧
主題：用簡單語言解釋複雜概念
提示：選一個最近學的概念，假裝對一個 5 歲小孩解釋。說不清楚的地方就是你還沒真正理解的地方，回去重新學習那個環節。
層級：理解層（Level 2）

### 星期三：知識連結
主題：建立知識間的橋樑
提示：今天學到的東西和你已知的什麼有關？試著找出一個因果連結、一個類比連結。孤立的知識容易遺忘，連結越多記憶越牢固。
層級：網絡層（Level 3）

### 星期四：間隔複習
主題：善用遺忘曲線的節奏
提示：艾賓浩斯遺忘曲線告訴我們：1天、3天、7天、14天是最佳複習間隔。今天花 10 分鐘複習一週前學的內容，效果勝過重讀三遍。
層級：記憶層（Level 1）

### 星期五：類比遷移
主題：跨領域應用所學
提示：把一個領域的知識遷移到另一個領域。例如：程式設計的「模組化」可以應用到寫作的「段落結構」。今天試著做一個跨領域類比。
層級：拓展層（Level 4）

### 星期六：刻意練習
主題：在舒適圈邊緣練習
提示：專業人士和業餘者的差別在於刻意練習。找一個你覺得「有點難但不至於挫敗」的挑戰，專注練習 25 分鐘。目標：現有能力 +4%。
層級：活用層（Level 5）

### 星期日：學習反思
主題：回顧本週學習成果
提示：問自己三個問題：(1) 這週學到最有價值的一件事是什麼？(2) 哪裡卡住了？(3) 下週要用什麼方法改進？反思是學習的加速器。
層級：整合

## 輸出格式

```
📚 今日學習技巧：【主題】
【提示內容】
— 《深度學習的技術》楊大輝
```
