# 每日摘要輸出格式

以下為摘要各區塊的排版模板。用實際資料替換描述文字。

---

🔄 連續報到第 N 天（由 digest-memory 提供）
- 昨日待辦：完成 M/N 項
- 習慣提示連續 N 天 | 學習技巧連續 N 天

📊 系統健康度（由 scheduler-state 提供，讀取 state/scheduler-state.json）
- 成功率 XX% | 平均耗時 XX 秒

✅ 今日待辦（由 todoist Skill 提供）
- 列出待辦事項

📰 本週屏東新聞（由 pingtung-news + pingtung-policy-expert 提供）
- 新聞標題
  → 政策背景：一句話解讀
  → 民眾有感度：N/10

🤖 AI 技術動態（由 hackernews-ai-digest 提供）
- 列出 AI 新聞標題、熱度與中文翻譯

💡 今日習慣提示（由 atomic-habits 提供）
- 一則《原子習慣》提示

📚 今日學習技巧（由 learning-mastery 提供）
- 一則《深度學習的技術》技巧

📝 知識庫回顧（由 knowledge-query 提供，若有）
- 列出最近相關筆記
- 本次匯入 N 則新筆記（若有匯入）

📬 今日郵件摘要（團隊模式限定，由 gmail 提供）
- 郵件標題列表

🔒 安全審查（團隊模式限定，由 skill-scanner 提供）
- 掃描結果摘要

☸️ 佛學禪語
- 一則禪語

🔧 Skill 使用報告
- 本次使用 N/12 個 Skill
- 快取命中：N 次 | API 呼叫：N 次 | 知識庫匯入：N 則
