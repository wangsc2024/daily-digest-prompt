# 實作計畫：八十八佛大懺悔文研究

## 概要
- **目標**：介紹八十八佛大懺悔文的歷史與意義，涵蓋文本溯源、經文結構、修行實踐、跨文化比較及現代應用等面向，形成一份 12,000–20,000 字的完整研究文件
- **預估任務數**：12 個（10 核心議題 + 導論 + 結論）
- **相關文件**：無既有規格，本計畫為全新研究

## 架構說明

本研究採**議題導向章節制**，每一核心議題對應一個獨立章節，便於分工撰寫與獨立驗收。全文依「歷史源流 → 文本分析 → 實踐面向 → 比較與展望」的邏輯遞進排列，確保讀者能從基礎知識逐步深入至當代應用與未來方向。

引用格式統一採用 **APA 第 7 版**，中文文獻依《學術論文寫作格式》慣例處理。

## 技術棧
- 語言：正體中文
- 格式：Markdown（最終可轉 Word/PDF）
- 引用格式：APA 7th Edition
- 工具：知識庫查詢（localhost:3000）、線上佛教文獻庫、學術資料庫

## 執行方式
使用 `executing-plans` skill 逐章執行此計畫
