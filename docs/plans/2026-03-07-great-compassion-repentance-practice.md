# 實作計畫：大悲懺法及其如何於生活中落實——系統性研究規劃

## 概要
- **目標**：系統性探討大悲懺法的歷史源流、儀軌結構、修持原理，並明確闡述如何將懺悔法門轉化為現代日常生活的實踐方法
- **預估任務數**：12 個核心研究任務 + 3 個整合任務 = 15 個
- **總預計字數**：12,000–15,000 字
- **相關文件**：CBETA 電子佛典、天台宗祖師著作、當代學術論文、修行開示文獻

## 架構說明

本研究採**歷史—義理—實踐**三層遞進架構：

1. **歷史層**（Task 1–2, 6）：溯源大悲懺的編制背景、經典依據與天台懺法傳統脈絡
2. **義理層**（Task 3–5, 7–8）：深入分析儀軌結構、懺悔理論、觀音信仰角色、修持禪觀要領及歷代感應
3. **實踐層**（Task 9–12）：從寺院現況到個人日常，完整涵蓋大悲懺與大悲咒的現代落實路徑，並與其他懺法進行比較

設計決策：
- 每個議題獨立成章，但透過交叉引用建立議題間的邏輯關聯
- 歷史與義理部分以原典文獻為主、學術論文為輔；實踐部分以當代開示與田野觀察為主
- 比較章節採對照表呈現，提升可讀性

## 技術棧
- 文獻來源：CBETA 電子佛典集成、祖師原典、Google Scholar / CNKI 學術資料庫
- 引用格式：中華佛教學會格式（輔以 APA 格式）
- 輸出格式：Markdown（可轉 DOCX / PDF）

使用 `executing-plans` skill 逐任務執行，搭配 `kb-research-strategist` 追蹤研究深化進度，完成後以 `knowledge-query` 匯入知識庫。
