# 法華經研究：方便品（開權顯實）— 三乘方便與一佛乘的真實意涵

## 研究主題與核心義理

本次研究聚焦《妙法蓮華經》第二品〈方便品〉，此品為全經「開權顯實」思想的樞紐。鳩摩羅什譯本中，世尊於本品明確揭示：過去所說聲聞、緣覺、菩薩三乘，皆為「方便」（upāya），真實唯有「一佛乘」。本品經題「方便」二字，即開權顯實之總綱——權者，應機施設之三乘教；實者，究竟圓滿之一佛乘。

**核心義理三要點：**
1. **開權顯實**：三乘是權，一佛乘是實；開除權教之執，顯發實相之真。
2. **諸法實相**：佛所證悟的「諸法實相」唯佛與佛乃能究盡，十如是（如是相、性、體、力、作、因、緣、果、報、本末究竟等）揭示法界全體。
3. **如來出世本懷**：令一切眾生開示悟入佛之知見，此為如來出現於世的唯一大事因緣。

---

## 重點章節解析

### 一、方便品 — 開權顯實的宣言

本品開篇，舍利弗三請、世尊三止，暗示此法人難信解。爾後世尊直示：

> 「舍利弗，如來但以一佛乘故，為眾生說法，無有餘乘，若二若三……諸佛如來言無虛妄，無有餘乘，唯有一佛乘。」（鳩摩羅什譯《妙法蓮華經》方便品第二）

此段確立全經基調：佛說法四十九年，阿含、方等、般若諸經，皆為導入法華一乘的階梯。天台宗判本品為「跡門」開顯之始，與〈壽量品〉開啟的「本門」形成法華二門結構。

### 二、譬喻品 — 三車火宅喻

緊接方便品之後，〈譬喻品〉以「三車火宅」喻深化開權顯實：長者宅起火，諸子貪著戲樂不知出離，長者以羊車、鹿車、牛車誘引，最後等賜大白牛車。三車喻聲聞、緣覺、菩薩三乘，大白牛車喻一佛乘。此喻揭示：三乘本是方便施設，真實目標唯一佛乘；眾生根機不同，故以三乘接引，終歸同一寶所。

### 三、信解品 — 窮子喻的自我認知轉化

四大聲聞以「窮子喻」領解：窮子畏怯逃離父城，父親以雇工身份漸次接近，最後付囑家業。此喻對應眾生本具佛性卻不自知，經由漸次教化終能荷擔如來家業。信解品從「領解」角度補足方便品的「開示」，形成「開示悟入」四字中的「悟」與「入」的實踐面向。

## 修行實踐要點

1. **不執權廢實**：日常修學中，不滯於阿含析空、方等彈偏，亦不耽般若但空，當知一切教法皆指向一佛乘。
2. **信受佛知見**：確信「一切眾生皆有佛性」「皆當作佛」，此信心為入法華之門。
3. **隨喜與流通**：聞法華經一句一偈，生隨喜心；更能展轉教化，與五十展轉功德相應。
4. **與止觀會通**：天台「一心三觀」「一念三千」皆以本品「諸法實相」為理論基石，持經者可結合《小止觀》修習。
