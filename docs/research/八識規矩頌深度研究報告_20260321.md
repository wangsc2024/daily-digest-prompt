## 摘要

本報告針對唐代佛教唯識學核心文本《八識規矩頌》進行系統性研究，涵蓋原文與譯文、歷史背景、核心概念、學術注釋比較、研究方法說明、結論與參考文獻。本研究採用 CBETA、DEDU、Wikipedia 及學術網路資源進行文獻檢索，交叉驗證多來源資料，並引用憨山德清、蕅益智旭、窺基、呂澂、周叔迦等至少五位學者觀點。報告指出：本頌雖傳統歸於玄奘，然近代學界（周叔迦、呂澂）對作者及年代存有爭議；文本以四章四十八句濃縮《成唯識論》要義，為唯識學入門關鍵；轉識成智的修行次第與當代認知科學、心理學有對話空間。

## 一、文本原文與譯文

### 1.1 原文（四章，四十八句）

《八識規矩頌》共四章，每章三頌十二句，七言體，合計四十八句。依 DEDU（數位佛典整合查詢）、CBETA 卍續藏 X0895 及朝陽科大佛學資源整理如下：

#### 第一章：前五識頌

```
性境現量通三性，眼耳身三二地居；
遍行別境善十一，中二大八貪瞋癡。

五識同依淨色根，九緣七八好相鄰；
合三離二觀塵世，愚者難分識與根。

變相觀空唯後得，果中猶自不詮真；
圓明初發成無漏，三類分身息苦輪。
```

#### 第二章：第六識頌

```
三性三量通三境，三界輪時易可知；
相應心所五十一，善惡臨時別配之。

性界受三恆轉易，根隨信等總相連；
動身發語獨為最，引滿能招業力牽。

發起初心歡喜地，俱生猶自現纏眠；
遠行地後純無漏，觀察圓明照大千。
```

#### 第三章：第七識頌

```
帶質有覆通情本，隨緣執我量為非；
八大遍行別境慧，貪癡我見慢相隨。

恆審思量我相隨，有情日夜鎮昏迷；
四惑八大相應起，六轉呼為染淨依。

極喜初心平等性，無功用行我恆摧；
如來現起他受用，十地菩薩所被機。
```

#### 第四章：第八識頌

```
性惟無覆五遍行，界地隨他業力生；
二乘不了因迷執，由此能興論主諍。

不動地前才捨藏，金剛道後異熟空；
大圓無垢同時發，普照十方塵剎中。
```

### 1.2 白話譯文（摘要式）

| 章節 | 要義 |
|------|------|
| **前五識** | 眼耳鼻舌身五識緣性境、現量，可通善惡無記；眼耳身三識僅在欲界與初禪二地現行；與三十四心所相應；轉依後成成所作智，能現大、小、隨類三種化身度眾。 |
| **第六識** | 意識具三性、三量、三境，相應五十一心所，為造業主體；初地初心初轉妙觀察智，七地後方純無漏。 |
| **第七識** | 末那識緣第八識見分為我，屬帶質境、非量；恆審思量執我，為染淨依；初地初心初轉平等性智，八地無功用行我執永伏。 |
| **第八識** | 阿賴耶識性唯無覆無記，與五遍行相應；異熟報由業力牽引；不動地前捨藏，金剛道後轉大圓鏡智。 |

## 二、歷史與典籍背景

### 2.1 出處與編纂時代

傳統說法據憨山德清《八識規矩通說》：「玄奘三藏糅譯《成唯識論》就，窺基法師因見本論十卷，文廣義幽，乃請法師集此要義，將八識分為四章，每章作頌一十二句。」故一般歸於玄奘（602–664），成書年代推測為七世紀中葉。

**爭議點**：周叔迦懷疑為晚唐所造，因唐人文獻及《宗鏡錄》均未提及；呂澂以頌文有文義瑕疵（如「非量」與「現量、比量」並稱三量、稱難陀為「愚者」等），認為非玄奘親筆。元代鮮于樞《困學齋雜錄》已錄有與第八識頌末八句幾近一致之文，顯示元代或更早已有流通。

### 2.2 所屬典籍

- **直接淵源**：《成唯識論》十卷（玄奘糅譯護法等十大論師釋）
- **藏經位置**：卍新纂續藏經 X55n0893、X55n0894、X55n0895 等
- **CBETA 編號**：X0895（蕅益智旭《八識規矩直解》）、X0893（憨山德清《八識規矩通說》）、X0894（廣益《八識規矩頌纂釋》）

### 2.3 與《大乘起信論》之關聯

憨山德清在《八識規矩通說》中引《大乘起信論》「眾生依心、意、意識轉」以會通唯心之旨，說明本頌可與如來藏系統對話，非唯識專屬。

## 三、主要概念解釋

### 3.1 八識結構

| 識 | 名稱 | 功能 |
|----|------|------|
| 前五識 | 眼耳鼻舌身 | 緣五塵，現量，性境 |
| 第六識 | 意識 | 分別、造業，通三量三境 |
| 第七識 | 末那識 | 恆審思量，執第八見分為我 |
| 第八識 | 阿賴耶識 | 藏種、持根身器界、異熟報體 |

### 3.2 三量、三境、三性

- **三量**：現量（親證）、比量（推度）、非量（謬執）
- **三境**：性境（實有）、帶質境（心緣心／心緣色）、獨影境（無質或假想）
- **三性**：善、惡、無記（含無覆無記、有覆無記）

### 3.3 轉識成智

| 識 | 轉成之智 | 關鍵階位 |
|----|----------|----------|
| 前五識 | 成所作智 | 五八果上圓 |
| 第六識 | 妙觀察智 | 初地初心、七地後純無漏 |
| 第七識 | 平等性智 | 初地初心、八地無功用行 |
| 第八識 | 大圓鏡智 | 金剛道後 |

### 3.4 與其他佛教教理之關聯

- 與《成唯識論》：「論雖十卷，其義盡此四十八句」（憨山德清）
- 與《百法明門論》：心所分配、五十一心所架構
- 與《大乘起信論》：心意識轉、業識現識等
- 與修行：參禪「離心意識參」與唯識觀心

## 四、學術評論與注釋

### 4.1 憨山德清（1546–1623）《八識規矩通說》

- **立場**：會通《起信》與《百法》，以「一真法界、圓明妙心」為大綱，強調「迷一心而為識」。
- **特色**：先提大綱，使學者易會；兼顧參禪者「不假廣涉教義，即此可以印心」。

### 4.2 蕅益智旭（1599–1655）《八識規矩直解》

- **立場**：依唯識論義逐句消文，詳釋性境、帶質、獨影及三量。
- **特色**：辨析「挾帶」「變帶」、根本智與後得智之差，破安慧等異師所計。

### 4.3 窺基（632–682）舊解

- **記載**：憨山稱「窺基舊解，以論釋之，學者難明，故但執相，不能會歸唯心之旨」。
- **地位**：唯識正統註疏之祖，本頌若出自玄奘，窺基當為最早傳承者。

### 4.4 呂澂（1896–1989）

- **觀點**：從文義與術語判定本頌非玄奘所作，指出「三量」並稱不當、對難陀之譏等。
- **意義**：開近代考據與義理並重之研究取向。

### 4.5 周叔迦（1899–1970）

- **觀點**：因唐人文獻與《宗鏡錄》未見記載，懷疑為晚唐託名之作。
- **意義**：提醒作者與年代需更多文獻佐證。

### 4.6 詮釋差異對照

| 面向 | 傳統（憨山、蕅益） | 近代（呂澂、周叔迦） |
|------|--------------------|-----------------------|
| 作者 | 玄奘 | 存疑，可能晚唐 |
| 目的 | 攝論歸心、修行印證 | 文本考據與義理檢視 |
| 會通 | 唯識×起信×禪 | 偏重唯識內部一致性 |

## 六、結論與啟示

### 6.1 在佛教思想體系中的意義

《八識規矩頌》以四十八句涵攝《成唯識論》核心，被譽為「一大藏教之關鑰」，是唯識學入門與修行觀心的重要綱領。其「轉識成智」次第與三性三量三境的分析架構，對理解心識結構、業力流轉與修行轉依具實質指導意義。

### 6.2 未來研究方向

1. **作者與年代**：結合敦煌寫本、日本古寫經與宋元藏經，進行更細緻的文本考古
2. **跨宗派比較**：與天台、華嚴、禪宗之心識論比較
3. **當代應用**：與認知科學、心理學的對話，如阿賴耶識與記憶、末那與自我感的比較研究
4. **譯本研究**：若有英、日譯本，可進行跨語境詮釋比較
