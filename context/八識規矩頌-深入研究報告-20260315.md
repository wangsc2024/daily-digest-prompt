## 一、研究摘要

《八識規矩頌》是中國唯識學的入門綱要文本，相傳為唐代玄奘三藏所造。全文四章十二頌四十八句（七字一句，四句一頌），與《大乘百法明門論》同為唯識學入門指南。本研究整合維基百科權威考證、專案知識庫洞見與既有報告，從作者爭議、傳播史、規矩九項、結構義理、轉識成智、註疏傳統、與 AI/認知科學的跨域同構等面向深化分析，並銜接專案唯識研究脈絡。

## 二、作者與文本定位

### 2.1 作者歸屬

- **傳統說法**：憨山德清《八識規矩通說》載，玄奘糅譯《成唯識論》後，窺基因該論「文廣義幽」，請玄奘集此要義，玄奘遂將八識分為四章、每章作頌十二句。歸於玄奘（602–664）所造，作為《成唯識論》撰成後的教學綱要。
- **學界質疑**：部分學者指出《八識規矩頌》不見於早期藏經目錄，文體與玄奘他作有別，疑為後人託名或宋代以後輯錄。
  - **周叔迦**：因普泰《八識規矩補註》（1511）為現存最早註，唐人著述及《宗鏡錄》皆未提及此頌，懷疑為晚唐所造，傳給後代失其人名而歸名玄奘。
  - **呂澂**：指出頌文文義瑕疵——將「非量」與「量」的兩類（現量、比量）並稱「三量」欠妥；又將難陀論師稱為「愚者」，判定頌文非出自玄奘之手。
- **文獻溯源**：元代鮮于樞《困學齋雜錄》錄有日休《八識詩》及第八識頌末八句，與今本《八識規矩頌》第八識頌幾近一致，顯示元代已有類似文本流傳。
- **實務定位**：不論作者考證結論如何，歷代註疏皆將其視為與《成唯識論》一致的正統唯識教法，在教學與實修中具有穩定地位。

### 2.2 在唯識典籍中的位置

| 文本 | 角色 | 與八識規矩頌的關係 |
|------|------|-------------------|
| 唯識三十頌 | 根本論 | 義理源頭，八識規矩頌為其濃縮綱要 |
| 成唯識論 | 註疏集成 | 玄奘糅譯十家釋，八識規矩頌可視為教學版提要 |
| 百法明門論 | 名相入門 | 百法與八識對應，八識規矩頌接續百法而說識體用 |
| 八識規矩頌 | 綱要頌 | 十二頌總攝八識體用與轉依，便於記誦與教學 |

## 三、結構與內容分析

### 3.1 十二頌的整體架構

《八識規矩頌》將八識分為四組，每組三頌：

| 組別 | 識 | 頌數 | 重點 |
|------|-----|------|------|
| 第一組 | 前五識（眼耳鼻舌身） | 頌 1–3 | 了境、性境、現量、五心、依根 |
| 第二組 | 第六識（意識） | 頌 4–6 | 分別、三量、三性、五十一名心所、三界九地 |
| 第三組 | 第七識（末那識） | 頌 7–9 | 思量、我執、恆審、四惑、轉依 |
| 第四組 | 第八識（阿賴耶識） | 頌 10–12 | 異熟、種子、執持、捨受、轉識成智 |

### 3.2 「規矩」的九項法度（凌波《八識規矩頌講記》）

「規矩」意為法度、軌則，涵蓋：

| 類別 | 內容 |
|------|------|
| 果用 | 八識各自所生的果報與功用 |
| 斷惑 | 各識所斷的煩惱與所知障 |
| 業用 | 造業與感果的機制 |
| 體相 | 各識的自體與行相 |
| 依緣 | 生識九緣（根、境、作意等） |
| 相應心所 | 五十一心所中與各識相應者 |
| 界地 | 三界九地中的活動範圍 |
| 三性 | 善、惡、無記 |
| 量與境 | 三量（現量、比量、非量）與三類境（性境、獨影境、帶質境） |

### 3.3 前五識頌（頌 1–3）要義

- **性境、現量、五心**：前五識緣「性境」（不帶名言推度之境），以「現量」直取，無分別。五心指率爾、尋求、決定、染淨、等流五個剎那心。
- **依根**：眼依眼根、耳依耳根等，隨根有無而識起滅，故有「眼識九緣生」等說法。
- **業用**：了別色聲香味觸，為意識提供感性資料，不直接造業但為業所依。

### 3.4 第六識頌（頌 4–6）要義

- **三量**：現量、比量、非量，意識具備推理與錯誤認識能力。
- **三性**：善、惡、無記，意識可造三性業。
- **五十一名心所**：與百法對應，意識與最多心所相應。
- **三界九地**：意識遍及欲界、色界、無色界，可生於九地之中。
- **業用**：分別、計度、造業主體，凡夫煩惱與修行轉依皆以此識為樞紐。

### 3.5 第七識頌（頌 7–9）要義

- **思量為性**：末那識恆緣阿賴耶識見分為「自內我」，執著為「我」。
- **四惑**：我痴、我見、我慢、我愛，與末那相應的根本煩惱。
- **恆審**：恆常審慮，不同於前六識的間斷。
- **轉依**：初地以上可轉為平等性智，但需至佛地方究竟。

### 3.6 第八識頌（頌 10–12）要義

- **異熟**：阿賴耶識為異熟識，承載業力種子，感招果報。
- **執持**：持種、持根、持身，為一切識與根身的所依。
- **捨受**：不與苦樂相應，唯是捨受，保持中性執持。
- **轉識成智**：轉阿賴耶識為大圓鏡智，成就佛果。

## 九、西方學術與英譯傳統（本輪研究補充）

### 9.1 Epstein 英譯與第八識詳解

Ronald Epstein（法界佛教大學 DRBU）的 *Verses Delineating the Eight Consciousnesses*（Tripitaka Master Hsuan-Tsang）為權威英譯本。其第八識頌釋義要點：
- **非覆無記**：第八識不覆蔽真如、亦不為心所覆蔽，被動不造善惡區分。
- **種子熏習**：前七識的業行「熏」第八識中的種子，使其成熟為現行；類比芝麻受花香熏染。
- **去後來先**：死時前七識攝入第八識，生前再生；中陰期間第八識「作主」。
- **不動地邊捨藏**：第八地前，第七識捨對第八識見分的執我，轉為平等性智。

### 9.2 禪宗四智與八識對應（Wikipedia）

禪宗核心教法之一為「轉八識成四智」：轉照向內，直見自性（見性）。對應關係：
- 第八識 → 大圓鏡智（Great Perfect Mirror Wisdom）
- 第七識 → 平等性智（Equality/Universal Nature Wisdom）
- 第六識 → 妙觀察智（Profound Observing Wisdom）
- 前五識 → 成所作智（All Performing Wisdom）

與《八識規矩頌》轉識成智完全一致，見惠能《六祖壇經》第七章、白隱《假名讀》、虛雲《開示錄》。

### 9.3 法相宗與華嚴宗的阿賴耶識詮釋

Thomas McEvilley 指出：法相宗與華嚴宗將唯識多個個體阿賴耶識收攝為「單一、普遍、恆常的宇宙阿賴耶識」，稱為「本初統一」（primordial unity），與 Vasubandhu 原說有別。
